# Acceso a la base de datos Flexygo App Offline

1. -Por un lado puedes ver el esquema de la base de datos dentro del diseño de la app, sobre la opción Esquema BBDD:

  

![](https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/2043125348536/original/qLAskMKCrMS0IuZ6SqXtzVFLY_WPOltCbw.png?1594137373)

  

En ésta sección podrás consultar el modelo de datos creado en la app, además de acceder a los asistentes correspondientes para realizar modificaciones

  

2. - Por otro lado una vez hayas realizado la primera sincronización, con las herramientas de Chrome pulsando 

f12 -> Application -> Web Sql tendrás acceso a la base de datos que se crea en cada dispositivo.

  

Con esta opción tienes acceso completo al modelo de datos de la app, tanto para ver cada una de las tablas, como para consultar datos y ejecutar instrucciones sql.

  

Adjuntamos link a la guía de uso y acceso al Web Sql integrado en Chrome.
